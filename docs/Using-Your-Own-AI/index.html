💬 Using Your Own AI

No OpenRouter key? No problem. Runico can hand you a ready-made prompt to run in any AI chat you already use — ChatGPT, Claude, Gemini — then turn the reply back into cards. It's completely free and needs no key and no network access from Runico.

This is the "Use my own AI" path on the Add screen. For the automatic alternative, see Using OpenRouter; for the whole card-drafting picture, see Adding Cards.

When to use this

  • You don't have (or don't want) an OpenRouter key.
  • You'd rather not pay per generation — your existing AI chat subscription already covers it.
  • Automatic generation in the browser hit a network / CORS error.
  • You simply prefer to see and tweak the prompt yourself.

How it works

Runico does the hard part — it writes a precise prompt and knows how to read the reply. You just shuttle text between Runico and your chat assistant.

  1. Start adding cards. In the browser, open a folder and begin adding cards to a topic (name it when asked, and drop in your source if you have one).
  2. Choose Use my own AI. Instead of running generation itself, Runico shows a ready-made prompt.
  3. Copy the prompt. Click Copy prompt.
  4. Paste it into your AI chat — ChatGPT, Claude, Gemini, anything. If your source is an image (a diagram or screenshot), attach that image in the chat alongside the prompt.
  5. Copy the reply. The assistant responds with card data; copy its whole response.
  6. Paste it back into Runico and click Use these cards.
  7. Review the drafts. Just like the automatic flow, you get Keep, Edit, or Remove for each card. Keep the good ones. ✨

💡 The prompt is carefully written so the reply is in exactly the format Runico expects — so copy the assistant's response as-is, without editing it, and let Runico parse it.

Image-occlusion cards from your own AI

If your source is a labeled diagram, Runico can also build an image-occlusion card (one that hides the labels for you to recall). There's an optional second prompt for this:

  1. Copy the occlusion prompt as well.
  2. Paste it into your chat with the image attached.
  3. Paste that reply back into Runico too.

Runico crops the figure and places the masks for you, ready to fine-tune in draft review. Either prompt alone is enough — whatever you paste back is combined into one set of drafts.

Tips

  • Any assistant works — there's nothing OpenRouter-specific here. Use whichever chat you like best.
  • Attach the image when your source is a picture, or the assistant can only work from text.
  • Don't reformat the reply. Paste the assistant's answer exactly as given; Runico is built to read that format.
  • If a reply doesn't import, ask the assistant to "return only the requested format," then copy and paste again.

A worked example

You're on Safari (no auto-generation) and have a ChatGPT tab open.

  1. In Runico you start adding cards to Cell signaling and choose Use my own AI.
  2. You Copy prompt, switch to ChatGPT, paste it, and hit enter.
  3. ChatGPT replies with a block of card data. You copy the whole thing.
  4. Back in Runico you paste it and click Use these cards — six drafts appear.
  5. You Keep five and Remove one. Done, for free. 🎉
