📚 Glossary

New to flashcards or to Runico? Here's every term used in this wiki, in plain English.

Card — a single flashcard. It shows a prompt with part of it hidden; you recall the hidden part, then reveal it to check. See Studying Cards.

Cloze — a card style where a word or phrase in a sentence is blanked out for you to fill in. Example: "The {{nucleus}} holds the cell's DNA." ("Cloze" just means fill in the blank.)

Q&A card — a card with a plain question on the front and the answer on the back.

Reversible card — a term-and-definition pair you can study in both directions (term → definition and definition → term).

Image-occlusion card — a card made from a picture with a label covered by a box; you recall what's hidden underneath. Great for diagrams. See Image-Occlusion Cards.

Topic — a single deck of cards about one thing, like Photosynthesis. Topics live inside folders.

Folder / subject — a container that groups topics (and other folders). Folders can nest as deep as you like. Biology is a folder; Cell Biology might be a folder inside it.

Library — your whole collection: all folders, topics, cards, and history. It's saved as one .runico file you own. See Your Library File.

Merge — folding one library into another: open one .runico file and pull a second one's folders, topics, cards, and history into it. Matching folders/topics combine; new ones are added. See Merging Libraries.

Duplicate (card) — during a merge, an incoming card that lands in the same place and has the same content as one you already have. Runico asks you to keep both, edit it, or skip it.

Session (or "sitting") — one round of studying. Runico records each session to chart your progress.

Due — cards that are ready for you to review now. A topic might say "4 due" meaning four cards are waiting.

Accuracy — for one session, the cards you got right ÷ the cards you reviewed, as a percentage. The progress chart plots this over time.

Action card — the summary panel that opens when you tap a topic in the browser, with its stats and buttons (Continue practice, Open cards, View progress…). See Browsing and Practicing.

Draft — an AI-generated card that's waiting for you to review (Keep or Remove) before it joins the topic. See Adding Cards.

OpenRouter — the service Runico uses to talk to AI models for drafting cards. You add a free API key from it in Settings.

API key — a private password-like string that lets Runico use AI on your behalf. It stays on your device.

Canvas / theme — the background color you study against: Light, Warm, or Dark.

File System Access API — the browser feature (in Chrome and Edge) that lets Runico auto-save straight into your .runico file. Browsers without it save by downloading.
