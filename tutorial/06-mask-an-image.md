# 6 · Mask an image

> Draw, drag, and resize boxes over a figure to build image-occlusion cards.

<p align="center">
  <img src="assets/occlusion.gif" alt="Mask an image — looping demo" width="700">
  <br><sub>Looping demo · plays automatically</sub>
</p>

Build **image-occlusion** cards by masking regions of a figure. Each box hides one label
for you to recall — perfect for diagrams, anatomy, maps, and any labelled image.

## Walkthrough

1. **Build image-occlusion cards by masking regions.** Each box hides one label to recall.
2. **Drag on the image to add a box.**
3. **Drag a box to reposition it.**
4. **Pull the corner to resize.**
5. **Remove a box with the × — keep only what matters.**

## What you see

- A figure (e.g. a cell diagram) with an editing surface on top.
- Mask boxes, each carrying a label tag (e.g. _Nucleus_, _Mitochondrion_) and a corner handle.
- An on-screen hint: **"+ Drag to add · drag to move · pull the corner to resize,"** plus a
  live region count (_1 region_, _2 regions_, …).
- Each box you keep becomes one occlusion card that hides that region's label.
